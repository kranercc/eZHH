# ez SSH ?

> How to use:
>* Put all your servers into `servers.txt` with the format: `ip:user:pass`
>* Execute `node main.js`

### Requirements

> Python:
>* paramiko (python3 -m pip install paramiko)
>
>Node:
>* express (npm install express)
>* body-parser (npm install body-parser)
